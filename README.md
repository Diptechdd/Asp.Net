# 🧠 CNN Image Classification (CIFAR-10)

This project implements a Convolutional Neural Network (CNN) for image classification using the CIFAR-10 dataset.

---

## 📌 Project Overview
- Built a deep learning model using PyTorch
- Applied data augmentation and regularization techniques
- Achieved **84.15% validation accuracy**

---

## 📂 Dataset
- CIFAR-10 dataset
- 60,000 images (32x32 color images)
- 10 classes:
  - Airplane, Car, Bird, Cat, Deer, Dog, Frog, Horse, Ship, Truck

---

## ⚙️ Technologies Used
- Python
- PyTorch
- NumPy
- Matplotlib
- Google Colab

---

## 🧠 Model Features
- Convolutional Neural Network (CNN)
- Batch Normalization
- Dropout Regularization
- Data Augmentation
- Learning Rate Scheduler

---

## 📊 Results
- Training Accuracy: ~85%+
- Validation Accuracy: **84.15%**

---

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Model will train and display results
